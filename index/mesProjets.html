Mes projets de fin d'années

GestMego

GestMego est une application client lourd permettant de gérer la collecte des mégots de cigarette sur la ville de Laon.

Elle est développée en C# avec l'IDE Visual Studio. La base de données est hébergée par MySQL

Fonctionnalités : Gestion du matériel de collecte, des lieux, des collectes, des techniciens.

Telem

Telem est un site de E-Commerce de piéces d'éléctroménager.

Il est développé en PHP avec le framework Symfony et le SCSS pour le visuel. La base de données est hébergée par MySQL

Fonctionnalités : Affichage, ajout, suppression, modification et recherche des pièces d'éléctroménager.
